# Facilitator Notes (starter)

Suggested run sequence:
1. Session Design: select composition group, scenario pack, and information regime
2. Facilitator Mode: record moves (actor, BGC domain, move type, action, rationale, confidence)
3. Governance Indicators: review aggregate metrics and threshold alerts
4. Foresight Validation: compare outcomes to selected historical baseline
5. After-Action Review: generate findings and export session artefacts
